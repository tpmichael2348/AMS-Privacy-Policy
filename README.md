# AMS 支援網站

這個資料夾包含 AMS 應用程式的支援網站和隱私權政策頁面。

## 文件說明

- `index.html` - 應用程式支援頁面
- `privacy.html` - 隱私權政策頁面

## 如何啟用 GitHub Pages

1. 將這些文件推送到 GitHub repository
2. 前往 GitHub repository 設定頁面
3. 點選左側選單的「Pages」
4. 在「Source」區域選擇：
   - Branch: `main` (或您的主分支)
   - Folder: `/docs`
5. 點選「Save」
6. 等待幾分鐘後，您的網站就會發布在：`https://yourusername.github.io/AMS-IOS/`

## 在 App Store Connect 使用

啟用後，您可以在 App Store Connect 填寫：

- **支援 URL**: `https://yourusername.github.io/AMS-IOS/`
- **隱私權政策 URL**: `https://yourusername.github.io/AMS-IOS/privacy.html`

## 自訂內容

請記得更新以下資訊：

### index.html
- 將 `support@example.com` 改為您的 email
- 將 `yourusername` 改為您的 GitHub 使用者名稱

### privacy.html
- 將 `support@example.com` 改為您的 email
- 將 `yourusername` 改為您的 GitHub 使用者名稱
