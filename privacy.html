我們重視並保護您的隱私

最後更新日期：2025 年 11 月 2 日

感謝您使用 DoremiAMS 資產管理系統。本隱私權政策說明我們如何處理您的資訊。

1. 資料收集與使用

1.1 我們收集的資訊

DoremiAMS 是一款主要在本地運作的應用程式。我們不會收集、傳輸或儲存您的任何個人資料到我們的伺服器。您在應用程式中輸入的所有資訊，包括但不限於：

  • 資產資料（金額、類型、名稱等）
  • 投資組合資訊
  • 財務目標設定
  • 個人備註

這些資料僅儲存在您的 iOS 裝置本地，使用 Apple 的 Core Data 框架進行安全儲存。

1.2 資料的使用目的

您輸入的資料僅用於以下目的：

  • 在應用程式內顯示您的資產資訊
  • 生成財務報表和圖表
  • 計算淨資產、投資損益等統計數據
  • 提供個人化的財務分析功能

2. 網路連線使用

雖然 DoremiAMS 主要是離線應用，但以下功能需要網路連線來提供即時資訊：

2.1 股票價格更新

當您使用「更新股價」功能時，應用程式會向第三方股票報價服務發送請求以獲取最新股價。這些請求中僅包含股票代碼，不包含您的個人資訊或資產數量。

我們使用的股票報價服務：

  • Finnhub API（用於美股報價）- 需要 API 金鑰
  • Yahoo Finance API（用於台股報價）- 免費公開 API

2.2 匯率更新

為了計算美股資產的台幣價值，應用程式需要取得最新的美元對台幣匯率。此功能：

  • 僅在您主動更新匯率時使用網路
  • 不會自動在背景連線
  • 不會傳送任何個人資料

重要說明：

  • 這些網路請求僅包含股票代碼或貨幣代碼，不包含任何個人資料
  • 您可以選擇完全不使用這些功能，手動輸入股價和匯率
  • 所有其他功能（記錄資產、查看報表、成就系統等）都完全離線運作

3. 資料安全

3.1 本地儲存

您的所有資料都儲存在 iOS 裝置的安全沙盒環境中，其他應用程式無法訪問。

3.2 生物辨識鎖定

應用程式提供 Face ID / Touch ID 鎖定功能，您可以在設定中啟用此功能以增強隱私保護。生物辨識資料由 iOS 系統管理，我們無法存取您的生物辨識資訊。

3.3 備份與 iCloud

如果您啟用了 iOS 的自動備份功能（iTunes 或 iCloud 備份），您的應用程式資料可能會被包含在備份中。這些備份由 Apple 管理，受到 Apple 隱私政策的保護。

4. 資料分享

我們絕不會向任何第三方出售、交易或以其他方式轉移您的個人資料。您的資料完全屬於您，並且只存在於您的裝置上。

5. 匯出與刪除資料

5.1 資料匯出

您可以隨時透過應用程式的「設定」→「匯出資料」功能，將您的資料匯出為 Excel 檔案。

5.2 資料刪除

您可以隨時在應用程式內刪除個別資產記錄。如果您想完全刪除所有資料，可以直接刪除應用程式，所有本地資料將被永久移除。

6. 未成年人隱私

本應用程式適用於所有年齡層。由於我們不收集任何個人資訊，因此不存在未成年人資料保護的相關問題。

7. 第三方服務

除了股票報價服務外，本應用程式不使用任何其他第三方服務或 SDK。我們不使用：

  • 分析工具（如 Google Analytics）
  • 廣告服務
  • 社交媒體整合
  • 追蹤服務

8. Cookies 與追蹤技術

本應用程式不使用 cookies 或任何追蹤技術。

9. 政策變更

如果我們對此隱私權政策進行任何重大變更，我們將在應用程式更新說明中通知您，並在此頁面更新修訂日期。我們建議您定期查看本政策以了解我們如何保護您的資訊。

10. 您的權利

作為應用程式使用者，您擁有以下權利：

  • 存取權：您可以隨時在應用程式內查看您的所有資料
  • 修改權：您可以隨時修改或更新您的資料
  • 刪除權：您可以隨時刪除個別記錄或整個應用程式
  • 匯出權：您可以將資料匯出為可攜式格式（Excel）
  • 停止使用權：您可以隨時停止使用本應用程式

簡而言之：您的財務資料完全屬於您，儲存在您的裝置上，我們無法存取，也絕不會收集或分享。僅在更新股價和匯率時需要網路連線，且不會傳送任何個人資料。
